# Pacman Game (Java OOP)
## 🎮 Description
This is a simple Pacman game built using Java and Object-Oriented Programming concepts.

## 🧩 Features
- Pacman movement using keyboard
- Ghost enemies with random movement
- Food collection system
- Score tracking
- Game over on collision

## 🎮 Controls
- `<` move left  
- `>` move right  
- `^` move up  
- `v` move down  
- `q` quit game  

## 🚀 How to Run
1. Open project in IntelliJ IDEA
2. Run `Game.java` file
3. Play using keyboard input
